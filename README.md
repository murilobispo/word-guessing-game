## About The Project
A Wordle-inspired desktop game built with Python and Tkinter.
Supports English and Brazilian Portuguese, with light and dark themes.

Created as a learning project to practice Python, string manipulation, and GUI development.

## Installation
Clone this repository and make sure you have Python installed.

1. Clone the repo
    ```sh
    git clone https://github.com/murilobispo/word-guessing-game
    ```
2. Install dependencies
    ```sh
    pip install requests
    ```
3. Run the game
    ```sh
    python app.py
    ```
